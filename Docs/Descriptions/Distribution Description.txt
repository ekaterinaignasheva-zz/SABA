Файл Distribution.txt содержит значения, описывающие 
поведение системы согласно условиям, указанным
в файле Parametres.txt. Стиль вывода этих значений так же 
указывается в файле Parameters.txt (см. Parameters_Description.txt)

В файле Distribution.txt данные хранятся ввиде таблицы: 
колонки (слева-направо): координата, импульс, энергия осциллятора;
строки соответствуют конкретному осциллятору в конкретный момент времени, 
всего кол-во_строк = кол-во_осцилляторов* кол-во_выбросов в файл.

Количество выбросов в файл и время, выбранное для выброса в файл,
указывается в файле Parameters.txt
