В файле Measures.txt данные хранятся ввиде таблицы:
колонки (слева-направо): 
время, 
момент первого порядка, 
момент второго порядка, 
число участия, 
индекс компактности, 
суммарная энергия трех центральных осцилляторов,
текущая полная энергия системы
ошибка по энергии

строки: моменты, соответствующие выбросу в файл 
(наглядно идентифицируются колонкой времени). 

Способ выброса в файл указвается в файле
Parameters.txt (см. Parameters_Description.txt)
